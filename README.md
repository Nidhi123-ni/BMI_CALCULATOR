# BMI Calculator

A simple and interactive BMI (Body Mass Index) Calculator developed using HTML, CSS, and JavaScript. This project allows users to calculate their BMI based on height and weight inputs and provides health category results instantly.

## Features
- Easy-to-use interface
- Instant BMI calculation
- Displays BMI category (Underweight, Normal, Overweight, Obese)
- Lightweight and beginner-friendly
- Clean UI design

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Enter your height
2. Enter your weight
3. Click the calculate button
4. View your BMI score and category
